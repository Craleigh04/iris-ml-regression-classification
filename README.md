# Iris Machine Learning Analysis (MLR & Logistic Regression)

This project applies multiple linear regression and multinomial logistic regression to the Iris dataset using IBM SPSS Modeler.

## Objectives
- Predict petal length using regression
- Classify iris species using logistic regression
- Evaluate model performance and overfitting

## Techniques Used
- Multiple Linear Regression (MLR)
- Multinomial Logistic Regression
- Feature importance analysis
- Model validation (R², p-values, Chi-square)
- Overfitting detection using train/test split

## Key Results
- MLR explained ~96% of variance in petal length
- Logistic regression achieved ~95% classification accuracy
- Minimal overfitting observed (~0.2%)
- Petal measurements were the strongest predictors

## Tools Used
- IBM SPSS Modeler
- Excel dataset
- Statistical analysis methods

## Why This Project Matters
This project demonstrates real-world machine learning workflows including model building, validation, and interpretation. These skills are directly applicable to data science, analytics, and AI engineering roles.
